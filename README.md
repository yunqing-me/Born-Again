# Installation and Environment:

- Platform: Linux
- NVIDIA A100 GPUs with CuDNN 11.4
- PyTorch>=1.3.0
- Python>=3.5.0
- lmdb, tqdm, wandb

Alternatively, you can install other libiraries through:  `pip install -r requirements.txt`
